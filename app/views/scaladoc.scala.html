ScalaUtils' Scaladoc documentation

ScalaUtils is released as part of the ScalaTest project. For every release of ScalaTest, there is a corresponding release of ScalaUtils with the same version number. Thus if you use both ScalaUtils and ScalaTest, you should make sure the version numbers match.

ScalaUtils Scaladoc documentation is integrated with ScalaTest's documentation. The links on this page link to the ScalaUtils portion of ScalaTest's Scaladoc documentation:

  • Scaladoc for ScalaUtils 2.0.RC1 (the latest 2.0 release candidate for Scala 2.10.0+)
