ScalaUtils 2.1.3 Release Notes

ScalaUtils 2.1.3 accompanies the ScalaTest 2.1.3 release. For more information, see the release notes for ScalaTest 2.1.3. For information on how to include ScalaUtils in your project, see the download page.

ScalaUtils 2.1.3 includes the following enhancements:

Enhancements

  • Enhanced Prettifier.default to handle scala.xml.NodeSeq and scala.xml.NodeBuffer.
  • Enhanced Prettifier.default to handle object that has class type Fred extends Seq[Fred].
  • Refactored Prettifier.default to use stringPrefix for better performance.

Please visit ScalaUtils Release Notes for previous version release notes.
